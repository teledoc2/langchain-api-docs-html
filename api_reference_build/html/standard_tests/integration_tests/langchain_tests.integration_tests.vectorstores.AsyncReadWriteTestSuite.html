AsyncReadWriteTestSuite#

class langchain_tests.integration_tests.vectorstores.AsyncReadWriteTestSuite[source]#

Test suite for checking the async read-write API of a vectorstore.

This test suite verifies the basic read-write API of a vectorstore.

The test suite is designed for asynchronous vectorstores.

Implementers should subclass this test suite and provide a fixture that returns an empty vectorstore for each test.

The fixture should use the get_embeddings method to get a pre-defined embeddings model that should be used for this test suite.
